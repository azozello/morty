Індивідуальне завдання №7

Для перевода чисел в из 10 в любую систему содаём новый класс NumberParser с приватными полями number и stringBuilder для храненние числа и его представления и public метод perseTenIn в которой с помощью рекурсси переводим число з 10 в любую систему исчисления. В методе класса Main считываем число и от нуля до этого числа в цикле используем метод parseTenIn в каждой итерации.


Індивідуальне завдання №8

В методе класса Main создаём анонимный класс, унаследованный от класса Runnable в котором переопределяем метод run. В нём с помощью try-with ресурсов открывает поток чтения из фаила и оборачиваем его в класс обёртку BufferedReader. построчно читаем из фаила все данные в String переменную а затем из каждой строки выделяем методом split нужные числа и полученные данные в метод из задания 6. Запускаем поток с помощью wait-notify-notifyAll конструкции для избежания дедлока. Полученные результат выводим в консоль
